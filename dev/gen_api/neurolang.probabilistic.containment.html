neurolang.probabilistic.containment module¶

neurolang.probabilistic.containment.is_contained(q1, q2)¶

Computes if q1 is contained in q2. Specifically, for 2 non-recursive positive ∃ logic queries, without constants, computes wether q1←q2.

neurolang.probabilistic.containment.is_contained_rule(q1, q2)¶

Computes if q1 is contained in q2. Specifically, for 2 Datalog rules without constants, computes wether q1←q2.
